## Running the code

To run the project locally, follow these steps:

1. Install the dependencies by running `npm install`.
2. Start the development server with `npm run dev`.

## Deploying the website

To deploy the website, you can follow these general steps:

1. Build the project for production by running `npm run build`.
2. Choose a hosting service (e.g., GitHub Pages, Netlify, Vercel).
3. Follow the specific instructions of the hosting service to upload your built files, typically found in the `dist` or `build` directory.
4. Ensure that your domain settings (if applicable) are configured correctly to point to your deployed site.

## Project Structure

The project consists of the following files and directories:

- `src/`: Contains the source code for the website.
  - `index.html`: The main HTML document.
  - `styles/`: Contains CSS files for styling.
  - `scripts/`: Contains JavaScript files for functionality.
  - `components/`: Contains reusable components like header, footer, and project cards.
  - `pages/`: Contains page-specific scripts.
- `package.json`: Configuration file for npm.
- `tsconfig.json`: Configuration file for TypeScript.
- `.gitignore`: Specifies files to ignore in Git.
- `README.md`: Documentation for the project.
